## vaq (Vim Awesome Query)

### Description

A CLI tool built to automate [vim-plug](https://github.com/junegunn/vim-plug) usage.

- Query/Search/Info for plugins with the vimawesome api
- List installed plugins
- Uninstall plugins
- Update plugins

### Install

`npm i -g @wallerbuilt/vaq`

### Use

`vaq`
